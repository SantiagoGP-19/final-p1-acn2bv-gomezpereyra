# 🎬 Sitio Web de Película

Aplicación web desarrollada como trabajo final de la materia Programación Web. El proyecto consiste en un sitio dinámico que presenta información sobre una película, integrando frontend y backend con base de datos.

## 🚀 Tecnologías utilizadas

* PHP
* JavaScript
* HTML5
* CSS3
* SQL

## 📌 Funcionalidades

* Visualización de información de la película
* Contenido dinámico conectado a base de datos
* Interacciones en el frontend mediante JavaScript
* Gestión de datos desde el backend con PHP

## 🧠 Conceptos aplicados

* Desarrollo Full Stack
* Integración frontend y backend
* Manejo de bases de datos relacionales
* Separación de lógica y presentación

## 🎯 Objetivo

El objetivo del proyecto fue aplicar los conocimientos adquiridos en desarrollo web, creando una aplicación funcional que integre distintas tecnologías.

## ▶️ Estado del proyecto

Finalizado como trabajo académico.
