# 🎵 Sistema de Gestión de Álbumes

Aplicación web desarrollada como trabajo final de la materia Programación Web. El sistema permite gestionar álbumes musicales mediante operaciones completas de alta, baja y modificación, integrando frontend y backend con base de datos.

## 🚀 Tecnologías utilizadas

* PHP
* JavaScript
* HTML5
* CSS3
* SQL

## 📌 Funcionalidades

* Alta de álbumes (creación de nuevos registros)
* Edición de álbumes existentes
* Eliminación de álbumes
* Visualización dinámica de datos desde la base de datos
* Interacciones en el frontend mediante JavaScript

## 🧠 Conceptos aplicados

* Operaciones CRUD (Create, Read, Update, Delete)
* Desarrollo Full Stack (frontend + backend)
* Conexión a bases de datos relacionales
* Manejo de formularios y validaciones
* Separación entre lógica de negocio y presentación

## 🎯 Objetivo

El objetivo del proyecto fue aplicar los conocimientos adquiridos en desarrollo web, construyendo una aplicación funcional con persistencia de datos.

## ▶️ Estado del proyecto

Finalizado como trabajo académico.
